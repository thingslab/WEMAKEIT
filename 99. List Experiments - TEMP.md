# WEMAKEIT — Experiments & Projects: Global Full List (ESP32)

All projects and experiments use the ESP32 platform (students buy or borrow a single reusable ESP32 DevKit module). Each entry uses: Title — Learn — Platform — Output.

---

## Basic Experiments (Foundations)

### 1. Blinking an LED

**Learn:** Digital output, timing (delay)

**Platform:** Breadboard / ESP32 DevKit

**Output:** Single LED blink circuit

### 2. Reading a Potentiometer

**Learn:** Analog input (ADC), mapping values

**Platform:** Breadboard / ESP32 DevKit

**Output:** Variable LED brightness or analog readout

### 3. Driving an RGB LED

**Learn:** PWM control, color mixing

**Platform:** Breadboard / ESP32 DevKit

**Output:** Color-changing RGB LED

### 4. Driving Multiple LEDs

**Learn:** Multiplexing basics, current limiting

**Platform:** Breadboard / ESP32 DevKit

**Output:** LED patterns and sequences

### 5. Push Buttons

**Learn:** Debouncing, digital input handling

**Platform:** Breadboard / ESP32 DevKit

**Output:** Button-controlled actions

### 6. Reading a Photoresistor

**Learn:** LDR behavior, ADC

**Platform:** Breadboard / ESP32 DevKit

**Output:** Light meter or automatic night light

### 7. Reading a Temperature Sensor

**Learn:** Sensor calibration, ADC or digital sensor readout (e.g., DS18B20, DHT22)

**Platform:** Breadboard / ESP32 DevKit

**Output:** Temperature display or threshold alarm

### 8. Driving a Servo Motor

**Learn:** PWM servo control, position commands

**Platform:** Breadboard / ESP32 DevKit

**Output:** Controlled angular motion (arm, pointer)

### 9. Using a Flex Sensor

**Learn:** Variable resistor readout, mapping to movement

**Platform:** Breadboard / ESP32 DevKit

**Output:** Bend-to-control interface

### 10. Reading a Soft Potentiometer

**Learn:** Continuous position sensing, ADC smoothing

**Platform:** Breadboard / ESP32 DevKit

**Output:** Linear position input for projects

### 11. Using a Piezo Buzzer

**Learn:** Tone generation, frequency control

**Platform:** Breadboard / ESP32 DevKit

**Output:** Beeps, melodies, simple sound effects

### 12. Driving a Motor

**Learn:** Motor drivers, direction and speed control

**Platform:** Breadboard / ESP32 DevKit + motor driver (e.g., L298N)

**Output:** DC motor control with driver

### 13. Using Relays

**Learn:** Electromechanical switching, isolation

**Platform:** Breadboard / ESP32 DevKit + relay module

**Output:** Control of higher-voltage loads

### 14. Using a Shift Register

**Learn:** Serial-to-parallel expansion, efficient IO

**Platform:** Breadboard / ESP32 DevKit

**Output:** Drive many LEDs or inputs with few pins

### 15. Using an LCD

**Learn:** Character/graphic displays, libraries, data/command interface

**Platform:** Breadboard / ESP32 DevKit

**Output:** Textual or basic graphical output (menus, readouts)

### 16. Simon Says (Memory Game)

**Learn:** State machines, sequences, input/output integration, timing

**Platform:** Breadboard / ESP32 DevKit

**Output:** Simple memory game with LEDs, buttons, and buzzer

### 17. Traffic Light System

**Learn:** Finite state machines, sensor inputs, pedestrian controls

**Platform:** Breadboard / ESP32 DevKit

**Output:** Programmable traffic-light simulation with buttons/sensors

### 18. GAMING & FUN (ESP32 Game Console)

**Learn:** Graphics, game logic, input handling

**Platform:** ESP32 DevKit (first time - BUY MODULE)

**Output:** Complete game console (OLED, D-pad, speaker)

### 19. CONNECTIVITY (IoT LED Controller)

**Learn:** WiFi, web servers, APIs

**Platform:** ESP32 DevKit (reuse module)

**Output:** IoT LED controller with web/mobile interface

### 20. COMPUTER ARCHITECTURE / SYSTEMS (VGA Computer / Capstone)

**Learn:** Video signals, keyboards, terminals, system integration

**Platform:** ESP32 DevKit (reuse module)

**Output:** VGA computer (Tiny BASIC) and/or integrated smart hub features depending on capstone focus

### 21. THT Soldering Practice

**Learn:** Through-hole (THT) soldering techniques, working with kits, safety

**Platform:** Small PCB kit that integrates with ESP32 (daughter board) / ESP32 DevKit

**Output:** Soldered THT kit (e.g., LED flasher or power supply) that connects to ESP32 headers

### 22. SMD/Complex IC Soldering & Assembly

**Learn:** Fine-pitch soldering, reflow or hand-soldering SMD components, mounting an IC (e.g., op-amp or shift register) onto a breakout/daughter PCB

**Platform:** SMD practice PCB + ESP32 DevKit integration

**Output:** Functional SMD board (e.g., amplifier, level-shifter, or small peripheral) the ESP32 can use

### 23. I2C Sensor Integration

**Learn:** I2C bus, addressing, pulling up lines, multiple devices

**Platform:** ESP32 DevKit + I2C sensors (BMP280, BME280)

**Output:** Multi-sensor readout (temperature, pressure, humidity) shown on display and logged

### 24. SPI Device & SD Card Logging

**Learn:** SPI bus, chip select, data rates, file I/O on SD cards

**Platform:** ESP32 DevKit + SD card module

**Output:** Data logger that stores sensor readings to an SD card

### 25. RTC & Persistent Storage

**Learn:** Real-time-clock modules (I2C/SPI), NVS/EEPROM usage on ESP32

**Platform:** ESP32 DevKit + RTC module (DS3231) + optional EEPROM/FRAM

**Output:** Time-stamped logs and persistent settings stored across power cycles

### 26. Bluetooth LE Communication

**Learn:** BLE peripheral/central roles, advertising, characteristics

**Platform:** ESP32 DevKit

**Output:** BLE-controlled device (e.g., remote control or sensor broadcaster)

### 27. Capacitive Touch Inputs

**Learn:** Capacitive sensing, touch thresholds, multi-touch arrays

**Platform:** ESP32 DevKit (built-in touch pins) + touch pads

**Output:** Touch-controlled UI (menu navigation, sliders)

### 28. Rotary Encoder & Encoder-Based Input

**Learn:** Quadrature decoding, debouncing, acceleration handling

**Platform:** ESP32 DevKit + rotary encoder

**Output:** Precise user input (volume knob, menu selector) for a project

### 29. Power Management & Battery Integration

**Learn:** LiPo battery charging circuits, regulators, battery monitoring (ADC), safe wiring

**Platform:** ESP32 DevKit + LiPo and charger module

**Output:** Portable powered project with safe charging and battery level reporting

### 30. Advanced Motor Control with Feedback

**Learn:** H-bridge control, PWM speed control, encoder feedback, basic PID

**Platform:** ESP32 DevKit + motor driver + encoder-equipped motors

**Output:** Closed-loop motor control for robotics projects

## Teaching Strategy

### Week 1: Unboxing & Basics

- Solder ESP32 Ultimate Platform

- Install Arduino IDE + ESP32 support

- Upload "Hello OLED" program

### Weeks 2-4: Guided Projects

- Instructor provides 10 pre-made programs

- Students upload, test, and modify parameters

### Weeks 5-8: Challenges

- Weekly programming challenges (e.g., alarm clock, game, weather station, security system)

### Weeks 9-12: Final Projects

- Students choose and combine features, add WiFi, present to class, share code
